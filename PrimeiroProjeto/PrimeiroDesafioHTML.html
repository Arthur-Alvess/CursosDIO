Tags H

Voltar

Existem várias variações da tag "H" no HTML, elas servem para modificar o tamanho da letra que será exibida, indo do h1 até o h6.

Elas funcionam da seguinte forma: a tag h1 é a que representa um texto maior, sendo utilizada para títulos, já a h2 para subtítulos. Dessa forma, quanto maior o número que está após o "H", menor será a letra.

Tag P

Voltar

A tag P serve para uma organização do seu texto, uma vez que ela gera separações no seu texto, o chamado parágrafo.

Tag OL

Voltar

A tag OL significa "Ordered List", ou seja, lista ordenada, ela serve para gerar listas ordenadas. Aqui está um exemplo:

  1. Bom dia!
  2. Boa tarde!
  3. Boa noite!

Tag UL

Voltar

A tag UL significa "Unordered List", ou seja, lista não-ordenada, ela serve para gerar listas não-ordenadas. Aqui está um exemplo:

  • Bom dia!
  • Boa tarde!
  • Boa noite!

Tag LI

Voltar

A tag LI significa "List Item", ou seja, item da lista, ela serve para gerar itens nas listas, sejam elas ordenadas ou não-ordenadas.

Tag A

Voltar

A tag A significa "Âncora", elá é utilizada para criar links dentro de uma página html, seja o link para algo exterior à página ou não. Aqui está um exemplo:

  • Este link te direciona ao início desta página HTML
  • Este link te direciona ao meu perfil do GitHub
